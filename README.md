🛒 E-Commerce Grocery Store Web Application
A full-stack E-Commerce Grocery Store Web Application developed using HTML, CSS, JavaScript, Python, and Django to provide a smooth and user-friendly online shopping experience.

🚀 Features
User Registration and Secure Login Authentication
Product Browsing with Category-wise Filtering
Add to Cart and Order Management System
Responsive and Interactive User Interface
Backend Development using Django Framework
Database Integration for Product and User Management
Optimized performance with 90% functional efficiency

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Python, Django
Database: SQLite / Django ORM

📌 Project Outcome
Successfully developed a scalable web application that enables efficient online grocery shopping with smooth navigation, secure transactions, and reliable system performance.
